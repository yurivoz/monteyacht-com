## Voice

Anti-AI rules — see [../BRAND-VOICE.md](../BRAND-VOICE.md). Key markers:
- Captain Alexey quoted by name only in his quote blocks
- Concrete numbers (€, hours, km, °C) instead of adjectives
- Local terms (Plavi Horizont, Žanjic, pršut) without translation
- Section "What's not usually written" admits inconvenient truths
- No clichés ("unique", "perfect", "unforgettable", "luxurious")
